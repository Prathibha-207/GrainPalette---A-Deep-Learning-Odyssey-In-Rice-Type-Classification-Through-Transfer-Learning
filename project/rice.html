GrainPalette

A Deep Learning Odyssey in Rice Type Classification through Transfer Learning

Project Overview

GrainPalette utilizes convolutional neural networks and transfer learning to accurately classify various types of rice grains. The model is trained using a pre-trained CNN backbone (like ResNet50 or EfficientNet) and fine-tuned on a curated rice image dataset.

Methodology

Leveraging the power of transfer learning, the project implements data augmentation, feature extraction, and classification techniques to distinguish between subtle grain features such as shape, color, and texture.

Technologies Used

  • Python & TensorFlow/Keras
  • Transfer Learning with ResNet50
  • OpenCV for Image Preprocessing
  • HTML/CSS for Project Presentation
